# PW3B-LPF
## Open source hybrid filter preamps for electric and upright bass, guitar, mandolin, and other musical instruments. Individual part SMD/part through hole modules for variable high pass filter, Baxandall bass/mid controls, and variable resonance/variable frequency low pass filter. Most modules can be configured for standalone use or in various mix and match combinations.

Important:

The resonant filter board can easily go into self oscillation and destroy itself and/or attached gear if built incorrectly or if a pot fails. It is highly recommended to bench test with an oscilloscope to verify stable operation over the full control sweep ranges before installation and use with an amp and speakers. Also note that many parts are in surface mount format, so this is probably best thought of as an advanced project best suited for very experienced builders. Little or no tech support is available, but the designs themselves are very well vetted over 9+ years of evolution. 

Manipulating input impedance is a good way of enhancing performance, and should be done in the first stage of the pedal or onboard preamp. In practice the usual default values for instrument amplification are often higher than those that give the best subjective results with these particular circuits. For bass guitar, front end shunt resistor values in the 220k-500k Ohm range have typically been the most popular choices in numerous listening tests I've conducted with various  musicians and electronics technicians, but input resistor value is strictly up to the end user to decide on. The same holds for piezo pickup electric-acoustic instruments, and in both cases lower input ressistor values often provide a better noise floor and less peaky interaction between pickup resonance and the induced LPF one. And then if you are using an instrument with an active preamp, of another pedal or signal source with a low output impedance, you may wnat to go considerably lower on input impedance, by say an order of magnintude or so. Again, your choice, but I strongly suggest an open minded approach. I've personally seen "must feed a 10Meg Ohm input impedance" pickups that sounded much better at 700k with one particular pedal format. But it's DIY and I'm not YOU. Have fun, above all else!
